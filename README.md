# lbx-telebot
bot for our small and nasty community

*Available commands:*

`/help` or `/h`
Show this help.

`/ver` or `/v`
Show the current version.

`/profile` [name]
Show the stored profile of the requester or another user.
Options:
      name - target chat participant.

`/top`
Show top users.

`/bottom`
Show reversed rating.
